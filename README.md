# 사용 방법

- 프로젝트 경로로 이동 후, 'npm install' 명령어를 입력하여 package.js에 명시된 모듈 자동으로 모두 설치
- 이후 'npm start' 명령어로 expo-cli 시작, 스마트폰에 expo 어플리케이션 설치
- 터미널에 뜨는 QR코드를 스마트폰으로 인식 후, 접속

## API 키 등 private 요소는 key.env 파일에 저장할 것
